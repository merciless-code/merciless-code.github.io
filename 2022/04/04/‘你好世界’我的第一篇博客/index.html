‘你好世界’我的第一篇博客

发表于 2022-04-04 更新于 2022-04-07 分类于 杂谈

前言

本文为随笔杂谈，聊聊搭博想法，无任何实质性内容！

经过三天的工作，把这个 Hexo - NexT 框架下的个人博客给搭建好了，还通过 github page（应该是这个吧？）在其域名下部署成功！Nice！不过目前而言还没有任何实质性内容，我也没有多少实质性知识能写，所以我决定写本文充数！=。=

为什么做博客？

感性的说是感觉好多人都在做，做起来应该挺酷的，而且看起来不难 确实也不难。理性的说可以把之前学过的知识在网上保存并分享，虽然也没多少人看，但至少我未来自己需要时会看我曾经怎么做的，这样知识就不会忘掉。

搭建时做了什么？

首先是 Hexo + NexT 的搭建，这两在 github 上收藏人数多更新活跃，而且简单！！！ 虽然但是Hexo有个反人类大坑… 其次是 markdown 语言，为超轻量级文本语言被人民喜爱？尽管我之前也没听说过… 我所采用的是 VScode + 插件 原因是 Typora 好像要钱而且 VScode 也足够轻量！最后是 git 和 github 的熟悉，至少我把 github 的密码密钥临时码（自译不是中译）给搞熟悉了，网络差害人啊！

之后打算把这三东西熟悉一下整理一下，这不就又有三篇博客可以水了吗？Nice！

博客经营目标

正如我 MerCode_blog 的网名，本博客内容是以分享编程语言及其扩展为主，但有趣实用的也不会放过。

  • 第一阶段的想法是先把之前弄过乱七八糟的东西整理发下博客，扩充下内容以方便美化下博客页面之类的（文章太少太寒碜），这一部分估计不会做的太详细，就是简单回顾，方便以后有需要能找到方法查资料。

  • 第二阶段主要以项目为主，我打算建一个专门放项目有关东西的 page 毕竟好不容易学点东西做出成品了，不写什么过不去的吧。还要有一个日记一样的 page 主要概括放一段学习的目标或者成果，要求简明清晰方便，估计需要安装什么插件还需要找找看。

  • 第三阶段是以人工智能领域相关知识为主，主要是从理论到实践发一些尝试的结果和我的理解。这一块是要重点工作的，但估计很难……

最终是搭建一个美观、丰富的博客，展示我大学期间的所学所想。不过目前的分类和规划还是太简单混乱，后期还要在慢慢改善。

以上水完！Over！
